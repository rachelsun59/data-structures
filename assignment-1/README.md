## Installation  Usage

**Requirements**
* Node 7.6 or higher 

### 1. Clone this repo

```bash
$ git clone https://github.com/rachelsun59/data-structures.git
```

### 2. Go to the `assignment-1` folder

```bash
$ cd assignment-1
```

### 3. Install the NPM dependencies

```bash
$ npm install
```

### 4. Run the `week1.js` file

```bash
$ node week1.js
```
