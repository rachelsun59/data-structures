## Installation  Usage

**Requirements**
* Node 7.6 or higher 
* Cheerio

### 1. Clone this repo

```bash
$ git clone https://github.com/rachelsun59/data-structures.git
```

### 2. Go to the `assignment-2` folder

```bash
$ cd assignment-2
```

### 3. Install the NPM dependencies

```bash
$ npm install cheerio
```

### 4. Run the `week2.js` file

```bash
$ node week2.js
```
